## 描述

划分区域的国家省市区级联选择

## API

### 属性
属性 | 说明 | 类型 | 可选值 | 默认值
---|---|---|---|---
region | 地区数据 | `Array` | - | -
title | 内容 | `String` | - | 选择地区
disabledCodes | 禁用的地区 `code` 数组 | `Array` | - | -
checkedCodes | 默认选中的地区 `code` 数组 | `Array` | - | -
showOverseas | 是否显示海外 | `Boolean` | - | `true`
plugAreaEnglish | 是否开启国际化地区插件 | `Boolean` | - | `false`

### Slot
属性 | 说明
---|---
default | -
